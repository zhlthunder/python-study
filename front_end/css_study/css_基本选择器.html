寻找标签的选择器总结：
1.class选择器
.c1{} //定义class
应用样式
2.标签选择器：
a{} //为所有的a标签定义样式，其它标签类似用法
但对于input标签，有特殊的用法：
input[type='text']{} //即为input标签中type=text的所有标签应用这个样式；
3.id选择器：
#uu{} //为id为uu的标签定义样式
//应用这个样式；
4.层级选择器，使用空格来标识层级，表示 谁下面的谁；
对于上面的html标签(使用了注释，请查看源码)，可以用 .c3 a div #a1{} ,通过绝对路径访问到span标签并应用样式；
5.组合选择器
a,p{} //为所有的a和p标签应用相同的样式；
