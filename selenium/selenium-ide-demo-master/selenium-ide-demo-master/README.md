## 项目结构

- 一级目录是 web 站点的名字，例如：baidu，是对百度搜索网站用 Selenium IDE 录制的自动化测试脚本
- 网站必须是互联网上能访问到的
- 脚本一律用 Selenium IDE 专用的格式，即：HTML 格式存放
- 脚本以功能点来组织测试案例
- 脚本录制的环境是 Selenium IDE 2.9, Firefox 49
